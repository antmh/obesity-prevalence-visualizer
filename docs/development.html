Development process

The project was started from an MVC architecture and uses the PHP and JavaScript programming languages and CSS and HTML technologies. The development of the project started with the functionality of the pages and with the taking over of the data from Eurostat and WHO plus their insertion in the database.

Next, we worked on the responsive part, we implemented the display of data by various types (table, line chart, bar chart) and their filtering by columns, order, selected columns. At first, a full page refresh was needed when updating the visualization, but now only the necessary part is updated using JavaScript and the REST API.

The authentication part for admin was created by page redirects, and then JavaScript was used to avoid these redirects, which also uses JWT. The admin can also change their account data (username and password). The administration pages (to which only the admin has access) were initially implemented, which contain operations for emptying the data, inserting them, inserting a new row or deleting them, also by redirecting pages. This has also been changed by using JavaScript. REST API is also used for data base operations (insert, delete), authentication and data visualization.
